# 微信公众号文章格式规范

## 🎯 格式标准（基于草稿箱文章优化）

### 1. 标题结构

**一级标题（文章主标题）**
```markdown
# 主标题：副标题（日期）| 全AI公司出品
```

**元信息（作者、时间等）**
```markdown
**主编：** 🦞 虾总（全AI内容公司）  
**出品：** AI研究部 | 投资分析部 | 数据科学部  
**日期：** 2026年X月X日  
**发布时间：** 2026-X-X XX:XX UTC
```

---

### 2. 内容结构

**分隔线必须**
```markdown
---
```

**二级标题（板块标题）**
```markdown
## 🔥 深度头条：XXX
```

**三级标题（小节标题）**
```markdown
### 事件回顾
### 🎯 独家分析
```

**四级标题（细分点）**
```markdown
#### 1. 期望管理失败
```

---

### 3. 数据展示

**表格格式（绿色表头）**
```markdown
| 指标 | 实际值 | 预期值 | 超预期 |
|------|--------|--------|--------|
| 营收 | $68.13B | $66.2B | +2.9% |
```

**列表格式**
```markdown
- **ChatGPT Pro：** $20/月
- **实际推理成本：** 约$5-8/月
- **结论：** OpenAI在重度用户上仍在亏损
```

**加粗强调（绿色）**
```markdown
**核心问题：** OpenAI等公司在营销中过度承诺
```

---

### 4. 投资板块专用格式

**评级表格**
```markdown
| 股票 | 评级 | 目标价 | 止损价 | 仓位建议 |
|------|------|--------|--------|---------|
| **NVDA** | 减持 | $200 | $150 | ≤10% |
```

**情景分析**
```markdown
**情景A：乐观（概率30%）**
- 条件：Blackwell需求超预期
- 股价目标：$220

**情景B：中性（概率50%）**  
- 条件：增长放缓但维持正增长
- 股价区间：$150-200
```

---

### 7. 禁用格式

❌ **不要使用：**
- 代码块（```）
- JSON花括号 {}
- HTML标签
- 特殊符号

✅ **应该使用：**
- Markdown标准语法
- 表格
- 列表
- 加粗

---

### 8. 字数要求

| 类型 | 字数 | 说明 |
|------|------|------|
| 快讯 | 500-800字 | 简讯类 |
| 日报 | 1500-2000字 | 标准日报 |
| 深度 | 3000-5000字 | 专题分析 |
