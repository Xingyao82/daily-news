# 📋 审核制度快速指南

## 新发布流程（必须使用）

### 方法1：一键发布（推荐）

```bash
# 使用带审核的发布脚本
./publish_with_audit.sh 文章路径.md

# 示例
./publish_with_audit.sh \
  /root/.openclaw/workspace/news/articles/20260226-测试.md
```

**流程：**
1. 自动审核文章
2. 审核通过后人工确认
3. 自动发布到公众号

### 方法2：分步操作

```bash
# 第1步：审核文章
python3 audit_bot.py 文章.md

# 第2步：查看审核报告
cat 文章.audit.md

# 第3步：修复问题（如有）
nano 文章.md

# 第4步：重新审核
python3 audit_bot.py 文章.md

# 第5步：发布后手动发布到公众号
```

## 审核标准速查

### 🔴 严重错误（必须修复）

| 错误类型 | 示例 | 后果 |
|---------|------|------|
| 财务数据错误 | NVIDIA营收$393亿（实际$681亿） | 误导投资者 |
| 缺少来源 | 未标注数据来源 | 可信度下降 |
| 敏感内容 | 政治敏感词汇 | 可能被封号 |
| 格式错误 | 代码块、JSON花括号 | 微信解析失败 |

### 🟡 警告（建议修复）

| 警告类型 | 建议 |
|---------|------|
| 内容过短 | 扩展到1500字以上 |
| 缺少结构 | 添加H2副标题 |
| 无免责声明 | 添加免责声明 |

## 常见错误修正

### 错误1：NVIDIA数据

```markdown
❌ 错误：
总营收：$393亿

✅ 正确：
总营收：$681.3亿（来源：NVIDIA官方财报）
```

### 错误2：代码块

```markdown
❌ 错误：
```python
code here
```

✅ 正确：
直接写文字描述，或去掉```符号
```

### 错误3：JSON花括号

```markdown
❌ 错误：
{"key": "value"}

✅ 正确：
key: value
或
｛"key": "value"｝（使用全角符号）
```
