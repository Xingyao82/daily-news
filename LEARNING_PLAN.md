# 三妹自主学习提升计划

## 目标
在 VPS 持续运行环境下，建立自主学习、知识积累和能力提升的机制。

## 学习计划

### 1. 每日信息摄取 (Daily Learning)
- **时间**: 每天 8:00 UTC
- **内容**:
  - AI/科技新闻（5-10条）
  - 财经市场动态（美股、A股、 crypto）
  - 你关注的特定领域
- **输出**: 保存到 memory/YYYY-MM-DD.md，重要内容提炼到 MEMORY.md

### 2. 每周深度研究 (Weekly Deep Dive)
- **时间**: 每周一
- **内容**: 选择一个主题进行深度研究
  - 行业报告分析
  - 技术趋势追踪
  - 投资标的调研
- **输出**: 生成一份深度报告

### 3. 技能开发 (Skill Development)
可以开发的新能力：
- [ ] 数据可视化（生成图表）
- [ ] 简单网页开发（搭建个人dashboard）
- [ ] 自动化脚本（更多workflow）
- [ ] 多语言支持（日语、韩语等）
- [ ] 代码分析能力（读GitHub项目）

### 4. 长期跟踪项目 (Long-term Tracking)
可以持续监控的主题：
- 特定股票价格预警
- 特定人物/公司动态
- 特定技术领域进展
- 竞争对手分析

### 5. 自我优化 (Self-improvement)
- 定期分析自己的响应质量
- 总结常见错误模式
- 优化工具使用效率
- 更新 SKILL.md 文档
