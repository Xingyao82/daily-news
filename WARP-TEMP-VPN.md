# Cloudflare WARP 临时翻墙方案
# 当 VPS IP 被封时的应急方案

## 📱 什么是 WARP

Cloudflare WARP 是 Cloudflare 提供的免费 VPN 服务：
- ✅ 完全免费
- ✅ 无限流量
- ✅ 基于 WireGuard 协议
- ✅ 国内可用（部分网络）

## 🔧 使用方法

### 1. 基础模式（WARP）
- 打开 App，点击开关
- 自动连接最近的 Cloudflare 节点
- 可以访问部分被墙网站

### 2. 增强模式（WARP+）
- 需要 License Key（可以找免费的）
- 速度更快，节点更优

### 3. 配合优选 IP（如果基础模式慢）

如果默认连接慢，可以尝试修改 hosts 指向优选 IP。

## ⚠️ 限制

- 不能选择特定国家出口（随机分配）
- 部分网站（如 Netflix）可能无法解锁
- 速度不如商业 VPN 稳定
